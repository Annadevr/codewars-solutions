<h2><a href=https://www.codewars.com/kata/5556282156230d0e5e000089/train/python target="_blank">DNA to RNA Conversion</a></h2><h3>8 kyu</h3><p>Дезоксирибонуклеиновая кислота, ДНК, является основной молекулой хранения информации в биологических системах. Он состоит из четырех оснований нуклеиновой кислоты: гуанина ("G"), цитозина ("C"), аденина ("A") и тимина ("T"). </p><p>Рибонуклеиновая кислота, РНК, является основной молекулой-мессенджером в клетках. РНК незначительно отличается от ДНК своей химической структурой и не содержит тимина. В РНК тимин заменяется другой нуклеиновой кислотой — урацилом ('U').</p><p>Создайте функцию, которая преобразует заданную последовательность ДНК в последовательность РНК.</p><p>Например:</p><pre><code>"GCAT"  =&gt;  "GCAU"</code></pre><p>Входная строка может быть произвольной длины, в том числе пустой. Все входные данные гарантированно корректны, то есть каждая входная строка будет состоять только из <code>'G'</code>, <code>'C'</code>, <code>'A'</code> и/или <code>'T'</code>.</p>
